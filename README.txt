This is a quick project that converts data from a legacy format to JSON format and shows the results in a webpage.

TO RUN:
	Open the solution in Visual Studio (developed in version 2015 Community) - may require admin rights.
	Rebuild the solution
	Select "IIS Express" or "Local IIS" under LegacyConverter.Web/Web project properties. "Local IIS" may require the user to create a virtual directory.
	Press RUN
	Optionally run tests
